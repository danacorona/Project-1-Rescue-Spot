# Rescue-Spot App

Create an app that searches for adoptable dogs in your area and pull in a map with dog parks in your area. App also includes resources for pet owners. 

## Built With 

* HTML
* CSS
* Materialize
* JavaScript
* Petfinder API
* Mapbox Api
